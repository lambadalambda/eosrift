# Eosrift Documentation

Eosrift is a self-hosted tunnel service with an ngrok-like CLI and workflow.

## Highlights

- HTTP, TCP, and TLS tunnels.
- SQLite-backed token and reservation management.
- Caddy-based HTTPS termination for web traffic.
- Docker-first deployment and integration testing.
